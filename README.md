# mr-bot

For the [Marble Race](https://store.steampowered.com/app/851640/Marble_Race/) official discord @ https://discord.gg/q29bPWE

## Setup

Clone the repo

```bash
git clone https://github.com/sarahkittyy/mr-bot.git
cd mr-bot
npm i
```

Create *and edit* the .env file

```bash
cp .env.example .env
```

## Starting

```bash
npm run start
```
